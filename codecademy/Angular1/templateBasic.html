Your First App

  • A module contains the different components of an AngularJS app.
  • ng-app directive to define the application scope.
  • ng-controller is a directive that defines the controller scope.
  • A controller manages the app's data.
  • Expression - {{}} | An expression displays values on the page
  • Filters - currency | date | uppercase (A filter formats the value of an expression)
  • ng-repeat directive - loops through an array and displays each element.
  • ng-src directive - sets the >img< element's src to a property in the controller.
  • Directives bind behaviour to HTML elements. When the app runs, AngularJS walks through each HTML element looking for directives. When it finds one, AngularJS triggers that behaviour. (like attaching a scope or looping through an array).
  • ng-click directive
  • $index - gets index
