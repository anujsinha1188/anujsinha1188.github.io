Styling with CSS

Colors

  • Foreground color - color, backgroung color - background-color
  • 147 named colors
  • RGB colors | rgb() | value - 0 to 255 | 16,777,216 possible colors | color picker
  • hex color codes | use base 16 | #FFFFFF == #FFF
  • HSL colors | hsl(_,_%,_%) | Hue(color, value 0 - 360) Saturation (amount of gray 0% - 100%) Lightness (amount of white 0% - 100%)
  • opacity | rgba() | alpha value | range 0 - 1 | similarly hsla()
  • redundant declaration - for multiple version of different browsers

Fonts

  • "type of font" - typeface | font-family: _; | default - Times New Roman | enclose in "" if more than one word
  • typography | 2 types - serif (Anuj | eg - Times New Roman, Georgia) sans-serif(Anuj | eg - Arial, Halvetica)
  • fallback fonts | font-family: _, _, _;
  • Google Fonts | <link href="https://fonts.googleapis.com/css?family=Raleway" type="text/css" rel="stylesheet"> | Raleway
  • font-size=_ | 3 units of measurement | px - pixel | em = width of letter m | % - default is 16px
  • line-height:_; | vertical spacing between lines | manipulating the leading
  • word-spacing:_; | default - 0.25em
  • letter-spacing:_; | kerning - technical term
  • font-wight:bold/normal;
  • font-style:italic; - italicize
  • text-transform:uppercase/lowercase;
  • text-align:left/center/right
